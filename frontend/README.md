# Canteen Review Frontend (Prototype)

This is a mobile-first React + Tailwind CSS prototype for the campus canteen review system.

## Routes

- `/` Home (random canteen recommendation)
- `/map` Food Map (list + heat columns)
- `/profile` Personal Center
- `/canteens/:id` Canteen Detail
- `/dishes/:id` Dish Detail
- `/admin` Admin Dashboard

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
